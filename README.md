##🏃🏻‍♂️ Getting Started :

```
# install dependencies
$ npm install

# development
$ npm start

# production mode
$ npm run build

# Format code
$ npm run format

```

##🥳 Features:

Cuisine Restaurant is full-stack web application build using React, NestJs, Postgres, Material Design System.

- Authentication
  - Login
  - Sign Up
  - Forgot Password
  - Reset Password
- Resources
  - Explore Menu
  - Explore Dishes
  - Infinite Loading
  - Filter Dishes
  - Dish Review
  - Search Dishes
  - Filter Dishes
- Payment
  - Stripe Payments
  - Order Invoice
- User
  - Add Dish To Cart
  - Favorites
  - Create Orders
  - View Orders
- Admin
  - Admin Panel
  - Change Order Status
  - Inventory Management
  - Multiple User Role
  - User Creation with different roles
